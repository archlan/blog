New themes for archlan/i3!

February 12, 2022 Â· One min read
Lanhild
Developer/Maintainer at ArchLan

A newer interface is now shipped by default with ArchLan.

Themes have been added with the option to choose between theme, using a simple shortcut. $kyoto+v (where $kyoto refers to "Ctrl") toggles the new ArchLan ThemeSwitch interface that allows you to choose between a multitude of varied and colorful themes.
