New themes for archlan/i3!

February 12, 2022 路 One min read
Lanhild
Developer/Maintainer at ArchLan

A newer interface is now shipped by default with ArchLan.

Themes have been added with the option to choose between theme, using a simple shortcut. $kyoto+v (where $kyoto refers to "Ctrl") toggles the new ArchLan ThemeSwitch interface that allows you to choose between a multitude of varied and colorful themes.

Lightdm doesn't start on the live ISO

January 9, 2022 路 One min read
Lanhild
Developer/Maintainer at ArchLan

When booting in the ArchLan live environment, it is possible that the lightdm display manager didn't start automatically. If it's your case, just run this command:

# The password for the liveuser is "archlan"
$ sudo systemctl start lightdm.service

This issue will be fixed in the next releases.
